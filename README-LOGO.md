## Logo Configuration

The ReCast logo uses these parameters:
- Mode: Wave Bars
- Seed: recast-identity
- Frequency: 5
- Amplitude: 80
- Complexity: 0.6
- Chaos: 0.15
- Damping: 0.85
- Bar Count: 80
- Bar Spacing: 2

These settings create the distinctive rainbow wave bar pattern that represents ReCast's generative nature.
